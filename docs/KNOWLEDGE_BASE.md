# 知识库编写规范

## 推荐文件格式

```markdown
---
title: 项目名称
city: 所属城市
category: 非遗类别
source_name: 来源机构
source_url: https://example.com
updated_at: 2026-07-01
---

# 项目名称

正文……
```

## 内容原则

1. 一份文件聚焦一个非遗项目、场馆或城市主题。
2. 标题和元数据使用明确名称，避免“资料一”“补充说明”等模糊命名。
3. 来源应优先选择政府、官方场馆、权威文化机构和公开名录。
4. 实时信息应写更新时间，并在正文中提醒用户再次核验。
5. 不将营销文案、未经证实的传说或模型生成内容当作事实资料。
6. 避免复制大段受版权保护文本，优先自行概括并保留来源链接。

## 检索行为

检索器综合 BM25、中文字符 n-gram、城市和领域词加权。清晰的标题、城市、类别和来源元数据会显著提高召回质量。
